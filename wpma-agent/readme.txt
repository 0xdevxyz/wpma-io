== Description ==

WPMA Agent ist das Herzstück der WPMA.io Management-Platform - eine KI-gestützte WordPress-Verwaltungslösung, die weit über einfaches Monitoring hinausgeht.

= 🤖 Autonomer KI-Agent für proaktive Wartung =

Der WPMA Agent analysiert kontinuierlich deine WordPress-Site und erkennt automatisch Probleme, bevor sie kritisch werden. Mit Claude & GPT-4 Integration erhältst du intelligente Handlungsempfehlungen und kannst Updates mit KI-gestützter Risikoanalyse durchführen.

= 📝 AI Content Hub & Publisher =

Generiere hochwertige, SEO-optimierte Inhalte direkt aus dem WPMA Dashboard und publiziere sie automatisch auf deiner WordPress-Site. Unterstützt mehrere Sprachen, Tonalitäten und integrierte Pexels-Mediensuche.

= 💰 WooCommerce Revenue Intelligence =

Für WooCommerce-Shops bietet WPMA detailliertes Revenue-Tracking mit technischer Event-Korrelation. Erkenne sofort, wenn technische Probleme (z.B. Downtime, Performance-Drops) deinen Umsatz beeinflussen.

= 🛡️ Enterprise Security & Performance =

* Tägliche automatische Security-Scans mit Vulnerability-Checks
* 24/7 Uptime-Monitoring mit sofortigen Alerts
* Core Web Vitals Tracking und Performance-Optimierung
* Trend-Analyse zur proaktiven Problemerkennung

= 🧪 Staging, Cloning & Backups =

Erstelle mit einem Klick sichere Staging-Umgebungen zum Testen von Updates. Automatische Full & Incremental Backups mit One-Click-Restore sorgen für maximale Datensicherheit.

= 👥 Team-Management & White-Label =

Perfekt für Agenturen: Verwalte mehrere Sites zentral, arbeite mit deinem Team zusammen und biete deinen Kunden ein vollständig gebrandetes Dashboard mit Custom-Logo und Farben.

== Installation ==

= Automatische Installation =

1. Suche im WordPress Admin-Bereich unter 'Plugins' > 'Installieren' nach "WPMA Agent"
2. Klicke auf "Jetzt installieren" und dann auf "Aktivieren"
3. Gehe zu 'WPMA Agent' im Admin-Menü
4. Melde dich bei app.wpma.io an (oder erstelle einen kostenlosen Account)
5. Kopiere den Setup-Token aus dem Dashboard
6. Füge den Token im Plugin ein und klicke auf "Verbinden"
7. Fertig! Deine Site wird jetzt automatisch überwacht

= Manuelle Installation =

1. Lade die Plugin-ZIP-Datei von wpma.io herunter
2. Gehe zu 'Plugins' > 'Installieren' > 'Plugin hochladen'
3. Wähle die heruntergeladene ZIP-Datei aus
4. Klicke auf "Jetzt installieren" und dann auf "Aktivieren"
5. Folge den Schritten 3-7 der automatischen Installation

= Nach der Installation =

Das Plugin startet automatisch mit:
* Stündlichen Health-Checks
* Täglichen Security-Scans
* Performance-Monitoring
* WooCommerce Revenue-Tracking (falls WooCommerce aktiv)

Alle Features kannst du im WPMA Dashboard unter app.wpma.io zentral verwalten.

== Frequently Asked Questions ==

= Brauche ich einen WPMA.io Account? =

Ja, du benötigst einen Account auf app.wpma.io um das Plugin zu nutzen. Die Registrierung ist kostenlos und du kannst mit einem Free-Plan starten.

= Ist meine Website sicher? =

Absolut. Wir übertragen keine Passwörter oder sensiblen Daten. Die Kommunikation erfolgt ausschließlich über sichere API-Keys und SSL-verschlüsselte Verbindungen.

= Welche Daten werden übertragen? =

* WordPress-Version, PHP-Version und Server-Konfiguration
* Plugin- und Theme-Status für Update-Management
* Performance-Metriken (Ladezeit, Core Web Vitals, Cache-Performance)
* Sicherheitsstatus (SSL, Debug-Mode, Admin-Nutzer, etc.)
* WooCommerce-Metriken (Umsatz, Bestellungen, Conversion-Rate) - nur bei aktiver WooCommerce-Installation
* System-Health-Checks und verfügbare Updates

Hinweis: Keine Passwörter, Kunden- oder Zahlungsdaten werden übertragen.

= Kann ich die Übertragung bestimmter Daten deaktivieren? =

Ja, in den Plugin-Einstellungen kannst du einzelne Monitoring-Features deaktivieren.

= Funktioniert das Plugin mit WooCommerce? =

Ja! Wenn WooCommerce installiert ist, aktiviert sich automatisch das Revenue Intelligence Feature. Du erhältst detaillierte Einblicke in Umsätze, Bestellungen, Conversion-Rates und Korrelationen mit technischen Events.

= Wie funktioniert der AI Content Publisher? =

Über das WPMA Dashboard kannst du KI-generierte Inhalte erstellen und direkt auf deine WordPress-Site publizieren. Das Plugin stellt dafür eine sichere REST API bereit, die HMAC-signierte Requests akzeptiert.

= Kann ich Staging-Umgebungen erstellen? =

Ja, über das WPMA Dashboard kannst du mit einem Klick Staging-Umgebungen auf .staging.wpma.io erstellen. Das Plugin erkennt automatisch, ob es auf einer Staging- oder Produktiv-Umgebung läuft.

= Ist das Plugin für Agenturen geeignet? =

Absolut! WPMA bietet vollständiges Team-Management mit rollenbasierter Zugriffskontrolle (Owner, Admin, Developer, Support, Client) und White-Label-Funktionen (Custom-Logo, Farben, Domain).

= Was passiert bei Plugin-Deaktivierung? =

Bei Deaktivierung werden alle Cron-Jobs gestoppt. Deine Daten bleiben in der WPMA.io Plattform erhalten. Bei erneuter Aktivierung wird die Überwachung automatisch fortgesetzt.

= Wie kann ich das Plugin komplett entfernen? =

Bei vollständiger Deinstallation werden alle lokalen Datenbank-Tabellen und Optionen gelöscht. Deine historischen Daten auf WPMA.io bleiben erhalten (kannst du im Dashboard löschen).
